# Выбиратор

Мини-сервис для сравнения товаров/услуг по разным категориям через интерфейс "барабана номеронабирателя".

## Структура репозитория

vybirator/
├── index.html              -> главный барабан со списком патронов (набор номера)
├── patrons/
│   └── vacuum.html          -> патрон "выбиратор пылесосов"
├── data/
│   └── vacuum_items.csv     -> данные по пылесосам (характеристики + расчётная стоимость уборки)
├── decision_log.md          -> журнал принятых решений по проекту
└── README.md

## Как это работает

- `index.html` — ядро барабана. Крутящийся диск + список патронов читаются из встроенного справочника `DIRECTORY`.
- Каждый патрон — отдельный HTML-файл в папке `patrons/`.
- Данные для патронов лежат в `data/` как CSV, редактируются через Google Таблицы и выгружаются сюда вручную.
- `decision_log.md` — история решений по архитектуре и структуре категорий, обновляется по мере обсуждений.
